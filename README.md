IoT Smart Waste Disposal System

Overview

IoT Smart Waste Disposal System is designed to monitor waste levels in real-time and improve waste collection efficiency.

The system helps reduce overflow issues by tracking bin status and supports better waste management using IoT-based monitoring and a web interface.

A credit-based reward system was also introduced to encourage user participation.

---

Features

- Real-time waste level monitoring
- Smart waste collection tracking
- Credit-based reward system
- Web interface for monitoring waste status
- Improved collection efficiency
- IoT sensor integration support

---

Tech Stack

- Python
- Django
- MySQL
- HTML
- CSS
- JavaScript
- IoT Sensors

---

Project Structure

- Frontend → User interface and monitoring dashboard
- Backend → Data handling and system logic
- Database → Waste data storage and tracking
- IoT Layer → Sensor-based waste monitoring

---

How to Run

1. Clone the repository
2. Open the project in VS Code
3. Configure MySQL database settings
4. Install Django dependencies
5. Run the Django server
6. Connect IoT sensor input for monitoring
